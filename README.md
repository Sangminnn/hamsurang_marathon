## 이런 게임이에요

함수랑 마라톤은 브라우저만 있으면 바로 시작할 수 있는 모바일 우선 멀티플레이 레이스 게임입니다.

별도 설치 없이 접속하면 닉네임 하나로 바로 참여할 수 있고, 짧은 한 판이 끝나면 보상을 받아 캐릭터를 꾸미고, 곧바로 다음 레이스에 뛰어들 수 있습니다.

## 어떻게 플레이하나요

1. **닉네임을 정하고 로그인합니다.**
   앱 설치도, 회원가입도 필요 없습니다. 닉네임 하나면 충분합니다.

2. **방을 만들거나, 친구가 보낸 룸 코드로 입장합니다.**
   활성 중인 방 목록에서 바로 들어갈 수도 있습니다.

3. **캐릭터 스킨과 트레일을 골라 출전 세팅을 마칩니다.**
   로비에서 준비 버튼을 누르면 모든 참가자가 준비되는 즉시 카운트다운이 시작됩니다.

4. **30초 스프린트 레이스!**
   좌우 버튼을 번갈아 눌러 캐릭터를 전진시킵니다. 같은 버튼만 누르면 효율이 떨어지니 리듬감 있게 번갈아 눌러야 합니다.

5. **순위에 따라 코인 보상을 받습니다.**
   1위 120코인, 2위 90코인, 3위 70코인, 그 외 50코인. 바로 다음 경기를 준비할 수도 있습니다.

## 캐릭터 소개

### 수랑이

둥글고 귀여운 실루엣의 메인 캐릭터입니다. 사랑스러운 반응과 다양한 코스튬 소화력이 강점이며, 누구나 쉽게 친해질 수 있는 입문자 친화 캐릭터입니다.

캡, 탐정, 레인보우, 메카닉, 선샤인, 익스플로러, 파머, 다이버, 블라썸, 헤드셋, DJ, 셰프, 우주비행사, 러너, 윈터, 파일럿, 뱅커, 캐딧까지 **18종의 스킨**을 수집할 수 있습니다.

### 함부기

느린 거북이지만 근성과 역전 드라마가 매력인 캐릭터입니다. 껍데기와 표정, 장비의 다양한 변주로 유쾌한 레이스 경험을 선사합니다.

해커, 분석가, 스트리머, 작가, 필름메이커, 그래피티, DJ, 우주유영, 셰프, 메달리스트, 소방관, 다이버, 가드너, 빌더, 페인터 등 **20종 이상의 스킨**을 수집할 수 있습니다.

## 꾸미기와 성장

레이스에서 획득한 코인으로 새로운 캐릭터 스킨(150~240코인)과 트레일 이펙트(110~150코인)를 해금할 수 있습니다.

트레일은 레이스 중 캐릭터 뒤에 남는 시각 효과로, 민트/스파크/플레임/페탈 4종 중에서 선택할 수 있습니다.

해금한 스킨과 트레일은 로비에서 자유롭게 교체 가능하며, 레이스 시작과 동시에 다른 참가자 화면에도 그대로 반영됩니다.

## 주요 특징

- **제로 설치** — 브라우저 접속만으로 바로 플레이
- **실시간 멀티플레이** — 룸 코드 하나로 친구와 즉시 합류
- **30초 한 판** — 짧고 빠른 경기로 부담 없이 반복 플레이
- **공정한 경쟁** — 서버가 모든 입력을 판정하여 치트 방지
- **모바일 퍼스트** — 세로 화면에 최적화된 조작감, 데스크탑에서도 쾌적

## 기술 스택

| 영역 | 스택 |
|------|------|
| 프론트엔드 | React 19, Vite, TypeScript |
| 게임 렌더링 | Phaser 3 |
| 게임 서버 | Node.js, Colyseus, Express |
| 실시간 통신 | WebSocket |
| 패키지 관리 | pnpm monorepo |

## 시작하기

```bash
# 설치
pnpm install

# 게임 서버 실행 (포트 2567)
pnpm dev:server

# 웹 클라이언트 실행 (포트 5173)
pnpm dev:web
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.

모바일에서 접속하려면 같은 Wi-Fi에서 `http://<PC IP>:5173` 으로 접속하세요.

```bash
# 외부 접속 허용이 필요한 경우
pnpm --filter web dev --host
```
